¿Qué es HTML?

Es el lenguaje de marcado estándar utilizado para crear páginas web y estructurar su contenido.

¿Qué es CSS?

Es un lenguaje de diseño utilizado para controlar la presentación y el formato de las páginas web.

¿Qué es JAVASCRIPT?

Es un lenguaje de programación ampliamente utilizado en el desarrollo web.
